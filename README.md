# ⚖️ BMI Calculator

A simple and responsive **Body Mass Index (BMI) Calculator** built with **HTML**, **CSS**, and **JavaScript**. It calculates the BMI based on the user's name, weight (in kilograms), and height (in centimeters), and classifies the result into health categories like Underweight, Normal, Overweight, and Obese.

---

## ✨ Features

- 🧍 User-friendly form to input name, weight, and height
- ⚡ Instant BMI calculation on button click
- 📊 Categorizes BMI into:
  - Underweight
  - Normal weight
  - Overweight
  - Obese
- ♻️ Reset button to clear all inputs
- 💻 Fully responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
